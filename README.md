# ScareBnB

ScareBnB is a clone of AirBnB. It was made with Express and React. It is an app you can use to rent out a bedroom or entire location to users.

## Features List

Features include: 
* GET / POST / PUT / DELETE A spot
* GET / POST / DELETE A booking

For more information click the [link](https://github.com/Irving-Develops/ScareBnB/wiki/MVP-Feature-List)

## Technologies Used

Backend:
* Express
* Sequelize

Frontend:
* React
* Redux
